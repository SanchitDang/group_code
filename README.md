# Group Code
"Group Code" is an Android App made using Flutter, and Backend is made using Firebase, which provides a secure and reliable platform for real-time data synchronization and authentication. \
GroupCode is a revolutionary new Flutter app that allows users to share their code with their group members. Whether you are working on a personal project or a team project, GroupCode provides a platform for real-time code sharing and discussion. With GroupCode, you can easily share your code with your group members, and they can provide instant feedback, suggestions, and solutions. GroupCode's user-friendly interface makes it easy to share your code and collaborate with your group members\
The app allows users to create and join groups, share code snippets, and chat with other members in real-time. With its built-in code editor, users can make changes to the shared code and see the changes instantly.



## Features
- Have Authentication with Firebase.
- Have all CRUD functionality.
- Groups can be deleted, renamed etc.
- Live, Real time Code Editor.
- Create/join different groups with your friends and share your code.
